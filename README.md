# Environment E

Environment E is monolithic kernel for i686 architecture.

### Checklist:

- MBv1 bootloader - DONE

- Serial I/O - DONE

- Ring 0 GDT - DONE

- IDT - working

- Ring 3 GDT - NO

- E1000 - NO

- IDE driver - NO

- File system - NO


***but... pay some attention. Some time, and THIS thing will ping your router, create files and directories, and read LBA sectors.***
