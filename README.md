# Environment E

Environment E is monolithic kernel for i686 architecture.

### Checklist:

- MBv1 bootloader - DONE

- Serial I/O - DONE

- Ring 0 GDT - DONE

- IDT - working

- Ring 3 GDT - DONE (i think no)

- Preemptive Multitasking - NO

- M-call - NO (THIS WILL BLOW YOUR BRAIN(I not promise))

- E1000 driver - NO

- IDE driver - NO

- File system - NO


***but... pay some attention. Some time, and THIS thing will ping your router, create files and directories, and read LBA sectors.***
